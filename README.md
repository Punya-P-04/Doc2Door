# 🩺 Medical Chatbot System

A Spring Boot based medical assistance application designed to provide basic healthcare-related support through a simple and interactive system architecture.

## 📋 Overview

This project focuses on building a healthcare-oriented backend application capable of handling user interactions, managing medical-related functionalities, and supporting chatbot-based communication workflows. The application is developed using Spring Boot and follows a modular structure for better maintainability and scalability.

## 🛠️ Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* REST APIs
* Maven
* PostgreSQL
* Git & GitHub

## ✨ Features

* Medical assistance chatbot workflow
* RESTful backend architecture
* Database integration using PostgreSQL
* Modular Spring Boot project structure
* API handling for healthcare-related interactions
* Clean backend organization using Maven
